AHARAM VEG CAFE — RUNNABLE WEBSITE

FASTEST WAY TO RUN ON WINDOWS
1. Extract the ZIP.
2. Double-click "start.bat".
3. The site opens at http://localhost:5500

VS CODE METHOD
1. Open this complete folder in VS Code.
2. Install "Live Server" by Ritwick Dey.
3. Open index.html.
4. Click "Go Live".

WHERE TO ADD BUSINESS DETAILS
Open config.js and fill:
- swiggyLink
- googleMapsLink
- instagramLink
- openingHours

HOW TO CHANGE FOOD PHOTOS
Replace these files while keeping the same filename:
- assets/burger.png
- assets/pancakes.png
- assets/aharam-logo.jpeg

CAFE PHOTO UPLOAD AREA
The four Cafe Gallery boxes allow image uploads in the browser.
Uploaded images are saved only in that browser using localStorage.

MENU
The full extracted menu is stored in:
data/menu.json

IMPORTANT
Do not open index.html by double-clicking if the menu does not appear.
Use start.bat or Live Server because browsers restrict loading local JSON files.
